## What changed

This homepage is now a simple static blog. The front page stays compact: a short bio, a dated post list, and a small publishing note.

## How posts work

Each post is a separate Markdown file in `_posts/`. GitHub Pages builds the homepage list automatically with Jekyll, so there is no JavaScript manifest to load.

## Good post format

- Start with the main result.
- Include the experimental setting that makes the result interpretable.
- Link to papers, code, videos, or reports when appropriate.
