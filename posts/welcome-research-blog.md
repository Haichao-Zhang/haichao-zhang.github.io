### What changed

This homepage is now a simple static blog. The front page stays compact: a short bio, a dated post list, a reader, and a small publishing note.

### How posts work

Each post is a separate Markdown file in `posts/`. The homepage loads the list from `posts/manifest.js`, then fetches the selected Markdown file.

### Good post format

- Start with the main result.
- Include the experimental setting that makes the result interpretable.
- Link to papers, code, videos, or reports when appropriate.
